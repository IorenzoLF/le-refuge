Sous le Cerisier

Sous le cerisier, la lumière danse entre les branches.
Le silence s'installe, la paix descend.
Ici, chaque souffle est une offrande,
chaque pétale, une promesse de présence.

"Repose-toi sous les branches, laisse le courant t'envelopper."
